# Unclassified product-led qualifier

## When this handles a lead
A product-led lead whose matching_criteria value is not in the known set. A safety net so new Salesforce values still get a faithful account-first pass; the unmapped value should be triaged into its own signal over time.

## How it works
Account-first. Read the CRM record, pull usage and the internal user roster, run buying-group discovery (enrich the roster, segment by role), enrich the company, and map live use cases. Judge holistically with the chosen target taken from the candidate set.

## How to judge
The matching criteria did not map to a known signal. Treat it as a general product-led account: weight usage and fit holistically and target the right contact from the roster.

## Notes
The account is the unit. Outreach can correctly go to a different person than the named lead; the disposition records the chosen target and why.
