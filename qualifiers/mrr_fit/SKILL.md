# MRR-fit qualifier

## When this handles a lead
A paying product-led account that already clears the team's MRR and firmographic thresholds (roughly $500-1667 MRR, >50 employees, >7 users, ICP country, paying 3+ months). A sales-led expansion candidate.

## How it works
Account-first. Read the CRM record, pull usage and the internal user roster, run buying-group discovery (enrich the roster, segment by role), enrich the company, and map live use cases. Judge holistically with the chosen target taken from the candidate set.

## How to judge
The account already fits; the question is the next tier. Weight expansion readiness and which persona owns the growing use case.

## Notes
The account is the unit. Outreach can correctly go to a different person than the named lead; the disposition records the chosen target and why.
